# Tanstack Query Demo

This project was created as a Tanstack Query demo to show basic usecases for work.

## Prerequisites

To run this project you need the following software installed.

- [Node 24](https://nodejs.org/en/download) ([nvm](https://github.com/nvm-sh/nvm) or [fnm](https://github.com/Schniz/fnm) is recomended)
- [pnpm](https://pnpm.io/)

## Technologies used

- [Vite](https://vite.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [Tanstack Start](https://tanstack.com/start/latest)
- [Tanstack Router](https://tanstack.com/router/latest)
- [Tanstack Query](https://tanstack.com/query/latest)
- [Tanstack Form](https://tanstack.com/form/latest)
- [Shadcn](https://ui.shadcn.com/) using [Base UI](https://base-ui.com/)
- [Lucide Icons](https://lucide.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Oxfmt](https://oxc.rs/docs/guide/usage/formatter.html)
- [Oxlint](https://oxc.rs/docs/guide/usage/linter.html)

Some of the code was generated with [Claude Code](https://www.anthropic.com/product/claude-code)

## Run locally

1. Clone repo
2. `pnpm install`
3. `pnpm dev`
